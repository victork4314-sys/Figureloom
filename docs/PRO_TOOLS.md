# FigureLoom Pro Tools

FigureLoom keeps specialist features behind one **Pro Tools** entry so the normal editor remains calm. The hub groups focused workspaces instead of adding every control to the ribbon.

## Arrange and group

- Shift-click, marquee, and touch multi-selection
- Multi-object movement and corner resizing
- Group and ungroup
- Align and distribute
- Smart alignment guides and edge snapping
- Anchored connectors that follow their source and destination objects

Grouped objects remain separate editable layers. Grouping stores a relationship rather than flattening the artwork.

## Data and charts

Paste CSV or tab-separated spreadsheet data to create editable tables and supported chart objects. Double-click a chart or table to reopen its source data, visual type, and title.

## Scientific annotation

Tools include panel labels, callouts, scale bars, measurement lines, significance and grouping brackets, numbered markers, legends, Greek symbols, superscript, subscript, and practical chemical notation.

For full TeX rendering, use the dedicated TeX typesetting workspace.

## Components and objects

- Reusable project components and instances
- Crop, flip, and mask uploaded images
- Union, intersection, and subtraction for native shapes

## Review and references

- Object and page comments with resolve states
- DOI, creator, source, license, and attribution records
- Automatic reference collection
- Named version checkpoints and change highlighting
- Alt-text drafting
- Contrast, tiny-text, grayscale, and color-vision previews

## Publish and present

- Generic publication-size presets
- Publication-readiness reports
- Fullscreen multi-page presentation
- Export diagnostics and compatibility reporting
- Editable SVG for the active page or every project page

Always verify the current requirements of the specific journal, conference, printer, or institution.

## Office bridge

- Import PPTX slides with supported text, shapes, images, tables, groups, and basic charts
- Import XLSX, XLS, XLSM, ODS, CSV, and TSV
- Insert editable tables and charts and retain supported workbook data

The current finished export panel does not create a PowerPoint file. Export checked SVG files and convert them in another presentation application when needed.

## Workspace and recovery

- Move or copy objects between pages
- Project asset management and broken-asset checks
- Rotating recovery snapshots
- Pre-export diagnostics
- Workspace reset and shortcut reference

## Advanced Science

Publication plot starters, sequence and protein-domain tracks, phylogenetic trees, gel and blot lanes, microscopy channel layouts, and specialist biological or environmental diagrams.

These are layout tools. Scientific analysis must still be validated in the appropriate specialist software.

## Accounts and gallery

- Local project gallery without an account
- Email and password accounts
- Confirmation and password recovery
- Owned and shared encrypted cloud gallery
- Explicit cloud saves
- Profile initials or scientific symbols

FigureLoom does not use Apple, Microsoft, or other social sign-in providers.

Cloud features remain disabled gracefully until [`CLOUD_SETUP.md`](CLOUD_SETUP.md) is deployed.

## Live collaboration

- Owner, editor, reviewer, and viewer roles
- Named presence and remote cursors
- Encrypted project broadcasts for editors
- Encrypted persistent review comments
- Email-account invitations
- Expiring guest links
- Guest join with a display name and no email account
- Optional 4 to 12 digit guest-link PIN
- Incoming-update pause while the local user is typing or dragging

## MCP and AI access

MCP is configured in Settings and uses the normal editor command registry.

- Project-specific private connection link
- Read-only or full editor access
- Separate destructive-action permission
- Current command discovery
- Normal Undo history and saving for successful writes
- Disconnect, reconnect, and revoke controls

MCP is optional and separate from Loomy.

## SVG path editor

- Inspect and edit SVG path commands and coordinates
- Select anchors on the canvas
- Break compound SVG artwork into independent editable SVG objects

This edits path command data. It is not a full Illustrator-style Bezier-handle suite.

## TeX typesetting

- MathJax TeX rendering to embedded SVG
- Editable retained TeX source
- Display or inline mode and color controls
- Double-click or inspector editing after insertion

MathJax loads on demand. Rendered equations are embedded into the project as vector artwork.

## Pathway exchange

Export the active page as starter structures for:

- SBGN-ML
- BioPAX RDF/XML
- SBML Level 3 XML

FigureLoom maps objects and connectors into machine-readable starter structures. Domain-specific validation in specialist pathway tools is still recommended.

## Loomy

Loomy can create an editable starting layout with the deterministic builder and optional providers where configured.

FigureLoom remains fully usable without Loomy or any external assistant.

## Interface philosophy

Advanced workspaces are progressively disclosed. The ordinary ribbon, illustration library, Add panel, canvas design controls, and Export menu remain usable without opening Pro Tools.
