# Quick task guides

Use this page when you know what you want to do and need the shortest reliable path. The longer [Complete tutorials](Tutorials) build full projects from start to finish.

## Start a clean project

1. Open Projects.
2. Choose New project.
3. Name the project immediately.
4. Choose the final page size or a close preset.
5. Download a `.figureloom` backup before a large import or major edit.

## Save a separate backup

1. Press Export.
2. Choose the editable FigureLoom project backup.
3. Save the `.figureloom` file somewhere outside the browser.
4. Keep dated copies before important revisions.

Local autosave is convenient, but a downloaded file is the safest portable copy.

## Recover a project after a reload or crash

1. Reopen FigureLoom in the same browser and device.
2. Open Projects and check the local gallery.
3. Open Recovery if the latest normal copy is incomplete.
4. Compare the recovery snapshot with the current project.
5. Download a backup as soon as the correct version is open.

Do not clear site data before checking recovery and the local gallery.

## Move a project to another device

1. Download the `.figureloom` backup on the original device.
2. Transfer that file normally.
3. Open FigureLoom on the new device.
4. Import the `.figureloom` file.
5. Confirm pages, objects, fonts, comments, and references.
6. Save a new local copy on the destination device.

## Add text that wraps into a block

1. Open Add.
2. Choose Text.
3. Paste or type the text.
4. Resize the text box width rather than stretching the letters.
5. Use the inspector to adjust font size, line spacing, alignment, and color.

Large pasted text should form a readable block instead of one endless line.

## Edit text directly on the canvas

1. Tap or click the text object.
2. Tap it again or press Enter when supported.
3. Edit the text in place.
4. Press Enter to finish or Escape to cancel.

Use the inspector when you need formatting rather than content changes.

## Select and move several objects

1. Turn on multi-select or hold the platform selection key.
2. Select each object.
3. Drag the selection together, or use exact position controls.
4. Group the objects when they should remain together.
5. Ungroup later when individual editing is needed.

On a phone, use the touch multi-select control instead of relying on a keyboard key.

## Move an object anywhere on the page

1. Select the object.
2. Drag from the object itself, not a resize handle.
3. Zoom in only for visibility. Zoom does not create new page area.
4. Open Edit and enter X and Y values for exact placement.
5. Check Layers if another object is intercepting the selection.

## Put an object in front or behind another

1. Select the object.
2. Use Bring forward or Send backward for one step.
3. Open Layers for full control.
4. Drag the layer to the required position.
5. Lock background objects after the order is correct.

## Duplicate or move an object between pages

1. Select the object.
2. Copy or duplicate it.
3. Switch to the destination page.
4. Paste or use the page transfer action when available.
5. Check the position because pages can use different dimensions.

## Add an image without stretching it

1. Open Add and choose Image or File.
2. Select the image.
3. Resize from a corner to preserve the aspect ratio.
4. Use crop or mask controls when the image needs a different frame.
5. Check image quality at the final export size.

Do not use FigureLoom as a replacement for scientific image analysis or raw-data processing.

## Add an editable SVG

1. Open Add and choose SVG or File.
2. Import the SVG.
3. Keep it as one editable vector object for normal use.
4. Use break-apart or path tools only when individual pieces need editing.
5. Check source and license metadata before publication.

## Create a chart from copied spreadsheet cells

1. Copy a clean rectangular range from the spreadsheet.
2. Open Charts or Add chart.
3. Paste the data.
4. Confirm headers and the delimiter.
5. Choose the chart type.
6. Add axis labels, units, and a legend where needed.
7. Double-click the chart later to reopen its data.

FigureLoom draws the chart. It does not validate the analysis or statistics.

## Add an equation

1. Open Add and choose Equation or TeX.
2. Enter the TeX expression.
3. Preview it.
4. Insert the rendered equation.
5. Resize it like vector artwork.
6. Keep the original TeX in project notes when future editing matters.

## Add a code window

1. Open Add and choose Code window.
2. Paste the code, command, or method example.
3. Choose the language label when available.
4. Turn wrapping on for instructions and long comments.
5. Keep literal code formatting when line breaks are meaningful.

## Make a study-site map

1. Open Map Studio.
2. Choose the geographic extent.
3. Add verified site coordinates.
4. Import GeoJSON when a boundary is needed.
5. Add labels, a legend, scale information, and source attribution.
6. Check whether sensitive locations should be generalized or omitted.

## Use Loomy without replacing existing work

1. Open Loomy.
2. Choose Gemini, Puter, or Builder when those choices are available.
3. Describe the structure, stages, direction, and required objects.
4. Generate the editable draft.
5. Inspect every label, asset, connector, and scientific claim.
6. Resize or rearrange the generated group so it fits inside the real page.

Puter does not use a FigureLoom quota. Provider availability and external service behavior can still vary.

## Switch between phone and desktop layout

1. Open Settings.
2. Choose Automatic, Desktop and tablet, or Phone.
3. Use Automatic for normal device-based behavior.
4. Force Phone when you want larger touch controls.
5. Force Desktop and tablet when a large tablet should keep all panels visible.

The project format and contents do not change when the interface mode changes.

## Export a transparent SVG

1. Set the project background to transparent where supported.
2. Check that no white rectangle is being used as a manual background.
3. Press Export.
4. Choose SVG.
5. Open the result in another browser or vector editor.
6. Confirm fonts, clipping, image placement, and transparency.

## Export a PNG for print

1. Set the final physical page size first.
2. Use the print PNG option.
3. Choose sufficient resolution for the destination.
4. Inspect the exported file at 100 percent.
5. Confirm dimensions before sending it to a printer or journal.

## Export to PowerPoint

1. Keep objects inside the page bounds.
2. Use common fonts when the presentation must open on another computer.
3. Press Export and choose PowerPoint.
4. Open the file in PowerPoint or another compatible application.
5. Check text wrapping, unsupported effects, charts, and image placement.

## Share a project for review

1. Sign in and save an encrypted cloud copy.
2. Open Share.
3. Invite the exact email address or create an expiring link.
4. Choose Viewer, Reviewer, or Editor according to the access needed.
5. Download a backup before a large collaborative session.
6. Resolve remote-update conflicts deliberately.
7. Revoke unused links after the work is complete.

## Run the final checks

1. Open Check.
2. Review off-page objects and broken connectors.
3. Check small text and contrast.
4. Add alt text and a long description where needed.
5. Confirm references and asset attribution.
6. Resolve comments.
7. Download the final project backup.
8. Export and inspect the result outside FigureLoom.
