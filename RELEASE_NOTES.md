# FigureLoom v1.0.0

This is the first published release of FigureLoom.

FigureLoom is a free, open-source editor for scientific figures, posters, diagrams, and presentations. It is built to stay useful without making AI the center of the workflow.

## Highlights

- Editable, page-based scientific figures with text, shapes, arrows, illustrations, layers, guides, alignment tools, and reusable templates
- Scientific asset libraries covering biology, microbiology, molecular work, chemistry, bioinformatics, maps, and more
- Charts, tables, code blocks, scientific symbols, rich text, and LaTeX support
- Project tabs, local autosave, recovery history, optional accounts, cloud projects, sharing, and collaboration
- Import and export tools for common figure and presentation workflows, including PowerPoint, PDF, SVG, and image output
- Desktop, tablet, and phone interfaces with light and dark themes
- Loomy as an optional helper for getting started, plus MCP access for compatible external assistants such as Claude
- MCP agents can work through FigureLoom's editor commands, show visible activity, and inspect the current page through screenshots

## Principles

- The editor works without an account for normal local use
- AI features are optional
- Projects remain editable instead of being flattened into generated images
