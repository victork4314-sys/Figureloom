# FigureLoom Bio Windows installer

Download and open `FigureLoom-Bio-Installer.exe`.

The installer is self-contained. It installs the language, local IDE, updater/repair window, automatic test launcher, desktop and Start Menu shortcuts, and an unzipped `FigureLoom Bio Test Files` folder. It also adds `flbio` to the current user's command path.

The installed shortcuts are:

- **Install or Update FigureLoom Bio**
- **FigureLoom Bio IDE**
- **Test FigureLoom Bio**

The installer runs the real CSV, FASTA, FASTQ, SVG, alignment, and phylogenetic-tree quick test before it finishes.
